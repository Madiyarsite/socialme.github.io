Gulp

Мен gulp препроцессорын қолданамын. Gulp html, css файлдарын өңдеп қысқартады. Және Sass, scss типті файылдарды css ке айналдырады.

Sass

CSS жазудың өзі қызықты, бірақ стиль кестесі үлкен болған кезде оны сақтау қиынға соғады. Бұл жағдайда бізге алдын-ала процессор көмектеседі. Sass CSS-тің өзінде қол жетімді емес функцияларды пайдалануға мүмкіндік береді, мысалы, айнымалылар, ұя салу, миксиндер, мұрагерлік және CSS жазу ыңғайлылығын қайтаратын басқа да жақсы нәрселер.

Webpack

Webpack қазіргі заманғы JavaScript қосымшаларына арналған статикалық модуль құрастырушысы болып табылады. Webpack сіздің қосымшаңызды өңдеген кезде, ол сіздің жобаңызға қажет әр модульді бейнелейтін және бір немесе бірнеше бумалар шығаратын тәуелділік графигін жасайды.

JQuery

jQuery - жылдам, кішкентай және бай JavaScript кітапханасы. Ол бірнеше браузерлерде жұмыс істейтін, қолдануға ыңғайлы API-мен HTML құжаттарымен жүру және манипуляциялау, оқиғалармен жұмыс, анимация және Ajax сияқты нәрселерді айтарлықтай жеңілдетеді. JQuery әмбебаптылық пен кеңейтудің үйлесімімен миллиондаған адамдардың JavaScript жазу әдісін өзгертті.
